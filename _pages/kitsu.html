Apa itu Kitsu Manga

Kitsu Manga adalah website untuk membaca manga secara online yang dirancang dengan fokus pada kemudahan penggunaan dan kecepatan akses. Dengan desain semi-transparan yang modern dan navigasi intuitif. Dibuat menggunakan Ruby on Rails.

Fitur di Kitsu Manga

  • Login/Register Auth dengan Devise
  • hCaptcha sebelum login/register
  • Role member and admin (guest bisa baca manga)
  • Email Confirmation Register dengan Action Mailer
  • UI Semi Transparent dengan Bulma
  • Pagination dengan Kaminari
  • Bookmarks Manga

Teknologi di Balik Kitsu Manga

Backend:
  • Ruby on Rails
    Framework Ruby on Rails dipilih karena kecepatan pengembangan dan skalabilitasnya. Rails juga mendukung integrasi gem (plugin) seperti Devise dan Kaminari dengan mudah.
  • SQLite (Default)
    Untuk pengembangan dan skala kecil, SQLite menjadi pilihan ringan. Namun, database bisa diubah ke MySQL atau PostgreSQL sesuai kebutuhan produksi.
Frontend:
  • Bulma CSS
    Framework CSS modern yang memudahkan pembuatan antarmuka responsif.
  • Stimulus.js
    Library JavaScript untuk meningkatkan interaktivitas.
Layanan Pendukung:
  • Action Mailer
    Mengirim email konfirmasi registrasi dan notifikasi.
  • hCaptcha
    Melindungi form dari serangan bot.
