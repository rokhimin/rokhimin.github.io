Apa Itu Atem Bot?

Atem Bot adalah bot serbaguna yang memungkinkan pengguna mencari informasi kartu Yu-Gi-Oh! dengan dinamis. Bot ini bisa diakses di tiga platform:

  • Discord
  • Telegram
  • WhatsApp

Alasan membuat Bot ini?

Karena saya sangat menyukai permainan kartu Yu-Gi-Oh! dan ingin memudahkan pengguna untuk mencari informasi tentang kartu-kartu tersebut. ditambah waktu itu saya sedang belajar membuat bot dengan discordrb (sebuah library di ruby untuk membuat bot di discord).

Tech di Balik Atem Bot

Atem Bot dibangun dengan bahasa pemrograman Ruby (70%) dan TypeScript (30%) dengan kombinasi library modern:

  • Ygoprodeck
    Digunakan untuk mengakses API dari Ygoprodeck, yang menyediakan informasi lengkap tentang kartu Yu-Gi-Oh! termasuk gambar, deskripsi, dan statistik.
  • Discordrb
    Library ini memungkinkan integrasi fitur seperti slash command dan embed message (websocket) dengan Discord.
  • Telegram-Bot-Ruby
    Untuk versi Telegram, bot menggunakan Telegram-Bot-Ruby, library yang mendukung pembuatan bot dengan fitur lengkap, termasuk handler pesan dan inline keyboard.
  • Baileys.js
    Karena WhatsApp tidak menyediakan API resmi yang free untuk bot (harus verify Document di Wahtsapp Bussiness), jadi saya memanfaatkan Baileys.js (library Node) untuk membuat koneksi WhatsApp Web yang stabil dan aman.

Dengan kombinasi teknologi ini, Atem Bot menjamin kompatibilitas dan kecepatan di semua platform.
